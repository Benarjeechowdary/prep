Front-End Developer Interview Questions

  • What is the difference between HTML, CSS, and JavaScript?
    HTML defines the structure of a webpage, CSS styles it, and JavaScript adds interactivity.
  • What are the different types of CSS positioning?
    The types are static, relative, absolute, fixed, and sticky positioning.
  • Explain the box model in CSS.
    The box model includes margins, borders, padding, and the content itself.
  • What is the difference between `==` and `===` in JavaScript?
    `==` compares values after type conversion, while `===` compares values and types.
  • What are pseudo-classes in CSS?
    Pseudo-classes are used to define the special state of an element (e.g., `:hover`, `:first-child`).
  • What is the purpose of `async` and `defer` in script tags?
    `async` loads the script asynchronously, while `defer` loads it after the HTML is parsed.
  • Explain the concept of responsive design.
    Responsive design ensures a webpage adapts to different screen sizes using media queries and flexible layouts.
  • What is the difference between `null` and `undefined` in JavaScript?
    `null` is an assigned value, while `undefined` means a variable has been declared but not assigned a value.
  • What is the purpose of the `z-index` property in CSS?
    The `z-index` property controls the vertical stacking order of elements.
  • What is the difference between `localStorage`, `sessionStorage`, and `cookies`?
    `localStorage` persists data with no expiration, `sessionStorage` persists until the tab closes, and `cookies` persist until expiration.
  • How do you improve website performance?
    Use lazy loading, minify CSS/JS files, enable browser caching, and reduce DOM size.
  • What are CSS media queries?
    Media queries allow you to apply styles based on the screen size or device type.
  • What is the Virtual DOM in React?
    The Virtual DOM is a lightweight representation of the real DOM used for efficient updates and rendering.
  • Explain the difference between `var`, `let`, and `const` in JavaScript.
    `var` has function scope, `let` and `const` have block scope, and `const` cannot be reassigned.
  • What is event delegation in JavaScript?
    Event delegation allows a parent element to handle events from its child elements using a single handler.
