Full-Stack Developer Interview Questions

Total Questions: 15

  • What is the difference between front-end and back-end development?
    Front-end development focuses on the user interface and experience, while back-end development handles the server, database, and application logic.
  • What is REST and how does it work?
    REST (Representational State Transfer) is an architectural style that uses HTTP methods to perform CRUD operations on resources.
  • What are the key differences between SQL and NoSQL databases?
    SQL databases are relational and structured, while NoSQL databases are non-relational and flexible with dynamic schemas.
  • Explain the concept of MVC (Model-View-Controller) architecture.
    MVC separates an application into three components: Model (data), View (UI), and Controller (logic).
  • What is the purpose of CORS (Cross-Origin Resource Sharing)?
    CORS allows or restricts resources on a web page to be requested from another domain.
  • What are microservices and how are they different from monolithic architecture?
    Microservices are small, independent services that communicate with each other, while monolithic architecture combines all components into one codebase.
  • Explain the difference between synchronous and asynchronous programming.
    Synchronous programming blocks execution until a task completes, while asynchronous programming allows other tasks to continue during execution.
  • What is an API gateway and why is it important?
    An API gateway handles requests and routes them to the appropriate microservice while managing authentication and rate limiting.
  • What is the difference between client-side and server-side rendering?
    Client-side rendering happens in the browser using JavaScript, while server-side rendering generates HTML on the server and sends it to the browser.
  • What are WebSockets and how do they work?
    WebSockets enable real-time communication between a client and server over a persistent connection.
  • What is the difference between `==` and `===` in JavaScript?
    `==` compares values after type conversion, while `===` compares both value and type.
  • What is JWT (JSON Web Token) and how is it used for authentication?
    JWT is a token used to securely transmit information between parties as a JSON object for authentication.
  • What are the advantages of using Docker for deployment?
    Docker provides containerization, which ensures consistent environments, easy scaling, and faster deployments.
  • What are Promises in JavaScript?
    Promises represent the eventual completion or failure of an asynchronous operation and its resulting value.
  • What is the difference between HTTP and HTTPS?
    HTTPS is the secure version of HTTP, using SSL/TLS encryption to secure communication between the client and server.
