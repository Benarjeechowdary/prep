Back-End Developer Interview Questions

Total Questions: 15

  • What is REST and how does it work?
    REST (Representational State Transfer) is an architectural style that uses HTTP methods (GET, POST, PUT, DELETE) to perform CRUD operations on resources.
  • What is the difference between SQL and NoSQL databases?
    SQL databases are relational, structured, and use a fixed schema, while NoSQL databases are non-relational, schema-less, and designed for scalability.
  • Explain the concept of middleware in web frameworks.
    Middleware functions are used to process requests between the client and server, handling tasks like authentication, logging, and request validation.
  • What are microservices?
    Microservices are an architectural style where an application is composed of small, independent services that communicate over HTTP or messaging protocols.
  • What is an API gateway?
    An API gateway manages and routes client requests to appropriate microservices, handling tasks like authentication, rate limiting, and load balancing.
  • What is the difference between synchronous and asynchronous programming?
    Synchronous programming waits for a task to complete before moving on, while asynchronous programming allows other tasks to continue without waiting.
  • What is dependency injection?
    Dependency injection is a design pattern where an object receives its dependencies from an external source rather than creating them internally.
  • What is the difference between TCP and UDP?
    TCP is connection-oriented and guarantees delivery, while UDP is connectionless and does not guarantee delivery.
  • Explain the purpose of caching in a web application.
    Caching improves performance by storing frequently accessed data in memory to reduce load times and server requests.
  • What is a reverse proxy and why is it used?
    A reverse proxy sits between client requests and backend servers, improving security, load balancing, and caching.
  • What is the difference between a monolithic and a microservices architecture?
    A monolithic architecture is a single codebase, while a microservices architecture breaks down functionality into small, independent services.
  • What is JWT (JSON Web Token) and how is it used?
    JWT is a compact, URL-safe token used to securely transmit information between parties as a JSON object.
  • What is the purpose of ORM (Object-Relational Mapping)?
    ORM allows developers to interact with a database using an object-oriented approach rather than writing raw SQL queries.
  • Explain the concept of load balancing.
    Load balancing distributes incoming network traffic across multiple servers to prevent overloading and improve response times.
  • What is the difference between vertical and horizontal scaling?
    Vertical scaling increases the capacity of a single server, while horizontal scaling adds more servers to handle increased load.
